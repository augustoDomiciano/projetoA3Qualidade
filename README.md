Descrição:

Esse projeto é focado no controle de frotas de veiculos, algumas das informações necessárias são id, placa e a marca. Foi implementado testes e integração com o banco de dados. Usamos a linguagem de programação Java, com o Spring Boot. O banco utilizado foi o MySQL.

Código adaptado das aulas do professor João Armenio.
